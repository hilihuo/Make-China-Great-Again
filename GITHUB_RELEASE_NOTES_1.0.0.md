# 中华人文史卷 v1.0.0

“中华人文史卷”以 26 个历史纪元为主线，通过人文图片、人物还原、事件还原、历史现场、可交互 3D 展品和中文语音讲解，构成可探索的中华人文历史长卷。

## 下载

### Windows 10/11 x64

下载：`China-Humanities-Museum-1.0.0-Setup.exe`

- 安装后可从桌面或开始菜单直接启动。
- 不需要安装 Node.js，也不需要运行本地服务器。
- 当前版本未使用商业代码签名证书，Windows 可能显示 SmartScreen 提示；请先核对 SHA-256。

### Android 7.0 及以上

下载：`China-Humanities-Museum-1.0.0.apk`

- 在手机或平板中允许文件管理器“安装未知应用”后安装。
- 推荐横屏使用，并预留至少 1.5 GB 可用存储空间。
- APK 使用项目独立发布密钥签名，已通过 v2/v3 签名验证。

## 主要功能

- 26 个历史纪元快速导航。
- 人文图片多图浏览。
- 人物、事件与历史现场图文还原。
- 可旋转、缩放和平移的 3D 展品。
- 展品参考图、细节说明和中文语音讲解。
- 各纪元独立背景声场。
- Windows 桌面安装版与 Android 横屏安装版。
